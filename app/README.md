Xyberwear is a cutting-edge **phygital fashion platform** that bridges physical apparel with the metaverse. Every physical product comes with a unique NFT, creating a seamless experience between reality and digital worlds.

## ✨ Features

### 🎨 Phygital Experience
- Every physical product includes a unique NFT
- Scan to unlock digital twins and exclusive content
- Cross-platform compatibility with major metaverses

### 🔧 Custom Design Studio
- AI-powered design generation
- Upload custom artwork
- Real-time preview

### 💎 NFT Integration
- Instant minting on purchase
- Multiple blockchain support
- NFT gallery and marketplace

### 🌐 Metaverse Ready
- Compatible with 10+ virtual worlds
- Wear your digital assets anywhere
- Cross-platform synchronization

### 📱 Modern Tech Stack
- Next.js 14 with App Router
- TypeScript for type safety
- Tailwind CSS for styling
- MongoDB for data persistence
- Express.js REST API
